Introduction

Lintel is the name of our support library. (A lintel is the thing that goes across a doorway to support the wall above it.) It's the place to put those miscellaneous pieces of code that could be potentially useful to others. The only caveats are:
  • Make sure that it should be in a library
  • Don't break the existing stuff!!!
  • Document what it does (here, and in the code) so others can use it

The Tcl/C++ interfaces

A system for the easy merging of Tcl and C++. You can think of it as either a method for easily using Tcl configuration scripts as a front end to your C++ system or for adding objects to Tcl.

There is a detailed usage guide.
